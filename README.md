  cos (alfa) = &lt;u,v> / (||u|| ||v||).


dados os vetores u, v de dimensão n:

calcule o ângulo que formam os vetores u e v.

A formula a ser utilizada é:

                    cos (alfa) = <u,v> / (||u|| ||v||).

alfa deve ser calculado em radianos e graus.

Ex1: supondo que n = 3

                 u= (1,2,3) e v=(-2,-4,2)

                 Alfa = 77,39562

Ex2: sendo n=3 

                  u= (-1,2,-3) e v= (2,-2,2)
                  Alfa = 157,792342
                  Alfa em graus = 22,207658
